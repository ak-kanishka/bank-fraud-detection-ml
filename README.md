# Bank Fraud Detection using Machine Learning

## Project Overview
This project is a machine learning based fraud detection system that predicts whether a bank transaction is safe or fraudulent based on transaction amount, device risk score, and IP risk score.

## Objective
To build an AI-powered fraud detection web application that can classify suspicious transactions and provide a simple explanation for the prediction.

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- XGBoost
- SQLite
- HTML
- CSS

## Features
- Fraud prediction using ML model
- Transaction amount input
- Device risk and IP risk input
- Fraud probability display
- Risk score calculation
- Explanation for prediction
- Prediction history storage
- Classic styled web interface

## Modules
1. Data Preparation
2. Model Training
3. Fraud Prediction
4. Explainability
5. Prediction History
6. Web Dashboard
